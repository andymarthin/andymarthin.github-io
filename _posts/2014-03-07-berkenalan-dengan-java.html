Belakangan ini saya lagi belajar java jadi sekalian aja nulis apa yang lagi sedang saya pelajari hehehe  Pertama tama berkenalan sama si Java. Bahasa Java diciptakan oleh James Gosling, Perancang system di Sun Microsystem Inc. Pada awalnya, James Gosling diberi tugas merancang software untuk berbagai peralatan elektronik berbasis mikroprosessor. Ketika mengunakan bahasa C/C++, ternyata program yang dibuatnya harus selalu dicompile ulang saat mengganti tipe dari chip mikroprosessor. Keadaan tersebut mencestukan ide untuk membuat satu bahasa yang bersifat "Write Once Run Anywhere " (WORA), yaitu bahasa yang memungkinkan program hanya sekali ditulis (compile) dan bisa dijalankan dimana saja pada chip apa pun.

pertama yang harus dilakukan adalah menginstall Java Development Kit (JDK) karena disini saya mengukanan ubuntu jadi disini saya tulis cara install JDK pada ubuntu bagi pengguna windows bisa cari caranya di google.

sudo apt-get install openjdk-7-jdk

setelah melakukan penginstallan JDK sekarang kita bisa memulai coding java. Seperti biasa saat baru mempelajadi suatu bahasa pemograman yang biasa saya lakukan ada membuat kalimat Hello Word terlebih dahulu

buatlah sebuah folder kosong untuk menyimpan file .java yang akan kita buat. misalnya /home/user/belajar-java setelah itu buka text editor kemuadian tulis kode berikut ini:

 

public class HelloWorld{
	public static void main(String[] args){
		System.out.println("HelloWorld!");
	}
}

Simpan dengan nama HelloWorld.java, ingat nama class harus sama persis dengan nama file jadi kalau nama classnya HelloWorld maka nama filenya harus bernama HelloWorld.java. setelah disimpan compile
file HelloWorld.java dengan Javac. Jangan lupa untuk memastikan sekarang berada di dalam folder yang sama dengan folder yang digunakan untuk menyimpan fle HelloWorld.java tersebut.

$ javac HelloWorld.java

Kalau proses kompilasi berjalan dengan baik, maka tidak ada pesan sama sekali dan di folder yang sama akan dibuat fle HelloWorld.class, setelah kompilasi selesai sekarang kita jalankan class HelloWorld.

$ java HelloWorld
