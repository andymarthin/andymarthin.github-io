Seperti yang Anda mungkin sudah tahu, SSH adalah sebuah protokol komunikasi yang aman yang memungkinkan Anda dari jarak jauh mengakses komputer jaringan. Hal ini dikenal sebagai pengganti Telnet yang sangat tidak aman. Sementara Telnet mengirimkan lalu lintas dalam teks biasa, SSH di sisi lain menggunakan protokol yang aman untuk berkomunikasi.

Buka Terminal, Tekan Â CTRL + ALT + T untuk membuka terminal kemudian jalankan perintah di bawah ini untuk menginstall SSH Server.

sudo apt-get install openssh-server

Itu saja! Gunakan SSH clients Anda untuk terhubung ke mesin anda menggunakan port default 22. Jika Anda ingin mengubah port koneksi, jalankan perintah di bawah ini untuk membuka file konfigurasi.

sudo gedit /etc/ssh/sshd_config

Kemudian mengubah port # terbukti apa pun yang Anda inginkan dan menyimpan file. Tapi ingat untuk menggunakan nomor port yang baru setiap kali Anda ingin terhubung ke sistem Anda melalui SSH
