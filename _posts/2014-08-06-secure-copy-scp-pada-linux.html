Secure Copy Protocol (SCP) adalah sebuah tool yang sangat berguna dalam mengirim dan menerima file melalui Secure Shell (SSH) di internet tanpa (atau dengan sedikit) resiko jalur kita terdeteksi oleh pihak lain. Dengan scp file yang akan disalin dikirimkan dalam bentuk chiper text (data yang dienkripsi). Berikut adalah cara menggunakan SCP untuk mengirim file dan menerima file.

Untuk melakukan penyalinan file/folder dari remote sistem ke lokal sistem dapat digunakan perintah :

Mengambil File/Folder

scp -r root@192.168.1.27:/var/www/project1 /var/www

Mengambil folder project1 dari remote folder /var/www yang ada pada IP 192.168.1.27 kedalam local directory /var/www

#  -r  digunakan bila kita akan mengirimkan sebuah folder, maka parameter “-r” akan dibutuhkan agar seluruh isi folder tersebut, termasuk file dan sub-folder juga ikut dicopy

 

Mengirim File/Folder

scp Document1.txt root@192.168.1.27:/backup

Mengirim file Document1.txt dari local directory ke remote folder sistem /backup yang ada pada IP 192.168.1.27
