Kenapa belajar java?

belakangan ini saya lagi belajar pemograman java, sebenarnya pengen belajar ruby tapi gk tau kenapa malah belajar java, dari dulu si emang ada niat mau belajar java tapi ya hanya niat dan tidak dilakukan. selama ini saya fokus di PHP. Di php juga masih newbie, kebetulan lagi pengen belajar konsep Object-oriented programming (OOP) tapi kebanyakan pake java yah uda sekalian aja belajar java.

Why?

berikut saya jabarkarkan kenapa saya belajar java dan kelebihan java.

  1. Gaji Programmer JAVA gede. Untuk starting pada saat tulisan ini dibuat, gaji fresh graduate JAVA start dari 4jt. Dan untuk gaji programmer JAVA EE yang senior gajinya >10jt. Kalo udah ngomongin duit pasti semangat.
  2. JAVA is Multi Platform, jadi Â java bisa dijalanin di beberapa OS yang sering dipakai seperti Windows, OSX dan LINUX.
  3. JAVA luas cakupannya, bisa develop desktop,web,bahkan mobile.
  4. JAVA is OOP. Kalau belajar JAVA, ente bisa sekalian belajar konsep OOP, nanti kalau mau belajar bahasa program lainnya yang pake konsep OOP ga bakal kesusahaan lagi.

yang bikin seneng saya belajar adalah dia multi platform, kebetulan saya sudah 2 tahun aktif menggunakan Linux jadi walaupun saya develop di linux bisa jalan juga di platform lain. mau develop desktop bisa, mau web? tinggal pake framework Spring, mobile? jangan tanya lagi Android bisa di develop mengukan java walaupun bisa didevelop bahasa lain tapi di internet lebih banyak tutorial develop android mengunakan dan komunitas java itu banyak banget men.
