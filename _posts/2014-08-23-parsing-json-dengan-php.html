JSON (JavaScript Object Notation) adalah salah satu struktur data JavaScript untuk mendefinisikan objek. Model struktur data JSON sekarang telah menjadi salah satu standar untuk pertukaran data yang ringan, yang sama populernya dengan XML.

JSON kini Â menjadi format pertukaran data baku yang terbuka ( open standart). Ukuran file JSON lebih kecil daripada file XML, karena pendefinisian yang lebih sederhana. Parsing data dari file JSON menjadi lebih cepat dan lebih mudah. hal inilah yang membuat JSON secara defakto menjadi alternatif format untuk pertukaran data.

Contoh JSON

{"post":[
		{"id":"1","nama":"Andy","jurusan":"Teknik Informatika"},
		{"id":"3","nama":"siapa aja","jurusan":"Desain Komunikasi Visual"},
		{"id":"4","nama":"Oline","jurusan":"DKV"}
	]}
