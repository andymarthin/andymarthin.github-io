FizzBuzz adalah permainan yang telah mendapatkan popularitas sebagai tugas pemrograman untuk menyingkirkan non-programmer selama wawancara kerja. Tujuan dari penugasan kurang tentang pemecahan dengan benar sesuai dengan aturan di bawah ini dan lebih lanjut tentang menampilkan programmer memahami dasar, alat yang diperlukan seperti if-/else-statements dan loop. Aturan FizzBuzz adalah sebagai berikut:

Untuk angka 1 sampai 100,

jika nomor tersebut habis dibagi 3 print Fizz;
jika nomor tersebut habis dibagi 5 print Buzz;
jika nomor tersebut habis dibagi 3 dan 5 (15) print FizzBuzz;
lain, mencetak nomor tersebut.

 

<?php 
	for ($i=1; $i <= 100 ; $i++) { 
		if (!($i % 15)) {
			echo "FizzBuzz <br>";
		}elseif (!($i % 5)) {
			echo "Buzz <br>";
		}elseif (!($i % 3)) {
			echo "Fizz<br>";
		}else{
			echo $i."<br>";
		}
	}
 ?>
