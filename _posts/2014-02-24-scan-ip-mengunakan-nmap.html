nmap adalah scanner powerfull yang tersedia di sistem Unix / Linux. Ini sangat berguna untuk eksplorasi dan audit keamanan jaringan. Ia dirancang untuk secara cepat melakukan scan terhadap jaringan yang besar.

 

jika nmap belum terinstal pada komputer anda silahkan menginstall nmap terlebih dahulu disini saya mengunakan ubuntu sebagai sistem operasi pada komputer saya. Untuk install nmap sangat mudah tingal mengikuti instruksi terminal di bawah ini

 

sudo apt-get install nmap

untuk melakukan scaning pada ip tingal mengikuti perintah di bawah ini

Scan ip dari 192.168.0.0 - 192.168.0.255

sudo nmap -sP 192.168.0.0/24

Scan IP dari 192.168.0.1 - 192.168.0.254

sudo nmap -sP 192.168.0.1-254

Scan Port dari Port 1 -139

nmap 192.168.0.1 -p 1-130

Scan Sistem Operasi pada IP target

sudo nmap -O 192.168.0.1

Scan Dengan cepat
jika kamu mau melakukan scan secara cepat tingal menambahkan option -T4 pada nmap

sudo nmap -A -T4 192.168.0.1

jika ingin lebih tau tentang nmap silahkan buka nmap manual

man nmap
